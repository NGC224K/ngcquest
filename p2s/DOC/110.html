범례

  • 입수 방법 "무기 가게의 소문"을 'A~E'로 표기, 시나리오 진행 단계는 '숫자'로 표기한다. (단계는 상점 정보를 참조)
    • A : 가격은 싸지만, 품질이 나쁨
    • B : 품질도 가격도 보통
    • C : 품질은 좋지만, 가격이 비쌈
    • D : 상품은 다양하지만, 매입가가 낮음
    • E : 상품은 부족하지만, 매입가가 높음
  • 입수 방법 "현상"의 소문은 A~C로 표기
    • A : 잘 맞지만, 대단한 것은 나오지 않는다.
    • B : 확률은 낮지만, 좋은 것이 맞는 것 같다.
    • C : 무턱대고 맞지는 않지만, 희귀한 것이 나온다.
  • 악마/마정 변화 : 악마/페르소나의 괄호내의 숫자는 레벨
    • 강조 표시의 악마는 레어 아이템으로 입수를 의미
