전작(죄)으로부터의 변경점

  • 로딩 시간이 대폭 줄어듬 (1/3 수준으로...)
  • 전투 고속화 : 전투 연출 설정에 '심플' 모드가 추가되어, 전투 연출(페르소나 소환 연출)을 스킵할 수 있다.
  • 컨택트가 1인 4커맨드에서, 1인 1커맨드로 간략화됐다.
  • 페르소나에 잠재 각성이 추가
  • 악마의 소문이 임의의 악마가 아닌, 특정 장소의 악마로부터 소문을 들을 수 있도록 변경

PS판과 PSP판의 차이점

PSP판이 이식작이므로 차이점은 PS를 기준으로 말합니다.
기본적인 차이점은 <PSP판 페르소나2 죄>와 동일합니다. 따라서, 나머지 차이점만 기재하겠습니다.
  • 신규 시나리오 추가 : 타츠야측 행동을 알 수 있는 시나리오가 추가됨
  • PS판 효과음을 그대로 사용 (PSP판 페르소나2 죄와의 차이점)
  • 일부 표기가 변경됨 (아래의 항목으로 설명)

클리어 데이터 인계

  1. 게임 엔딩 이후 데이터를 세이브한다.
  2. 타이틀 화면에서 'RESTART'를 선택
  3. 클리어 데이터를 선택해서 새 게임을 시작

이어받는 요소 (본편 → 본편 기준)

  • 타로 카드
  • 페르소나/악마/합체 마법의 애널라이즈
  • 판명된 식사의 효과
  • 카르마 링(カルマリング)
  • 시간성에 등록한 카드

2주차 이후의 특전

  • 이전에 플레이한 루트와 다른 루트로 클리어하면, EX던전이 개방된다. (세이브 데이터의 마크에 난조와 에리가 모두 표시)
  • 루시퍼(난조 루트), 사탄(에리 루트)의 합체 스킬 : '하르마게돈'을 사용 가능
  • 2종의 페르소나의 변이 사양을 변경 (아래의 표를 참고)

세이브 데이터의 인계

'페르소나2 죄'의 세이브 데이터를 '페르소나2 벌'에서 로드하면 다음과 같은 효과가 나타난다.

주인공(타츠야)의 이름, 레벨, 파라미터를 계승

레벨 55 이상의 데이터를 계승했을 경우 <이름, 레벨, 파라미터>를 그대로 이어받아서 등장한다. 다만, 장비와 페르소나는 인계되지 않는다.

카르마링을 처음부터 소지한 상태로 시작

전작(죄)의 폐공장에서 보상의 악마로 출현했던 앨리스(アリス)를 격파하면 입수할 수 있었던 카르마 링(カルマリング)을 처음부터 소지한 상태로 시작한다.
카르마 링의 효과 변경 : 공격력 +100 → STR+15

프리 타로의 개수를 이어받음

전작(죄)에서 가지고 있었던 '프리 타로'의 개수를 그대로 이어받는다.
예를 들어 전작에서 999매의 프리 타로를 가지고 있었으면, 벌에서 데이터를 불러와서 시작할 때 프리 타로를 999매 가지고 시작하게 된다.

전작의 선택에 따라 달라지는 요소

전작의 각종 선택 사항에 따라서 벌의 일부 이벤트에 영향을 준다.

전작 '보병궁의 신전' 클리어 이후에 벨벳룸에서, 에이키치의 질문에 "기타가 튄다 (ギターが弾ける)"라고 대답하면 <우라라+카츠야+타츠야>의 3인 컨택트 커맨드를 빠른 시점부터 사용할 수 있게 된다.

합체 스킬의 애널라이즈를 계승

전작에서 사용하여 애널라이즈 완료된 합체 스킬의 목록을 벌에 계승시킬 수 있다.
다만, 벌에서 일부 합체 스킬의 사양이 변경된 점에 따라 다소 차이가 있다. (삭제된 합체 스킬이나 성능의 변경 등등)

마네키 네코와 포피군에게 넣은 돈의 카운트가 계승

탐정 사무소에 있었던 '마네키 네코'와 코난 경찰서의 '포피군'에게 돈을 넣었던 카운트가 계승된다. (죄에서 카운트를 다 채워서 보수를 받은 경우는 초기화 상태)
