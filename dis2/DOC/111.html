방어구

　
  • IR : 아이템 랭크. 아이템계의 초기 레벨과 훔치기 확률 등에 영향.
  • 상품 : 상점에 진열될 가능성이 있는 상품 등급 범위.
  • HL : 표준 가격(레벨 0, 이노센트 없음, 레어리티 통상의 구입 가격). 매각의 경우 반액.
  • HP~RES : 레벨 0 의 보정치. Rare는 1.25배, Legend는 1.5배가 된다.
  • 이노센트 : 해당 아이템에 반드시 붙어 있는 이노센트의 종류
  • ※ 랭크 40의 아이템은 항상 레전드 등급이다.
