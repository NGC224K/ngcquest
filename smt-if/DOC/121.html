다크 악마는 무엇인가?

다크 악마는 악마 회화로 동료로 할 수 없는 악마이며 합체로 만들 수 있습니다.

다크(DARK)계에 속하는 종족은 다음과 같습니다.
사신(邪神) 요수(妖獣) 마왕(魔王) 악령(悪霊)
흉조(凶鳥) 요목(妖樹) 사룡(邪龍) 외도(外道)
사귀(邪鬼) 시귀(屍鬼) 유귀(幽鬼) -

다크 2신 합체

다크 악마의 레벨이 높은 경우
레벨의 합계치 합체 결과
2의 배수 다크 악마가 1랭크 UP

다크 악마의 레벨이 낮은 경우
레벨의 합계치 합체 결과
3의 배수 비(非)다크 악마가 1랭크 DOWN
5의 배수 비(非)다크 악마가 1랭크 UP
7의 배수 비(非)다크 악마가 2랭크 UP

다크 3신 합체

  • 다크 악마가 1체의 경우 : 다크 이외의 악마의 2몸합체 결과를 1랭크 UP 한 뒤 다크 악마와 합체시킨다.
  • 다크 악마가 2체의 경우 : 다크 악마끼리의 2몸합체 결과를 1랭크 UP 한 뒤 나머지의 악마와 합체시킨다.
  • 3체 모두 다크 악마의 경우 : 다크 악마 중에서 가장 레벨의 높은 것을 3랭크 UP 시킨다.
