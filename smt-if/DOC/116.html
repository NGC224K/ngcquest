레어 악마 설명

  • 레어 악마는 어느 특정의 포인트를 통과했을 때에 1/256 확률로 출현합니다.
  • 기본적으로 해당 악마의 레어 가디언을 가지고 있지 않아야 출현합니다.
  • 3명의 히어로들과 고토우는 유미, 레이코, 찰리 루트에서만 출현합니다.
  • 마인을 물리치면 강력한 아이템을 얻을 수 있습니다. (일부 한정)

레어 악마 목록

개 파스칼 (犬パスカル)

  • 상성 : 검, 총, 검 기술 이외 무효
  • 출현 조건 : 가디언이 파스칼이 아니다.
출현 후 선택지가 나오며 각각 상황이 변화한다.
  • 싸운다(たたかう) → 전투 개시
  • 도망친다(逃げる) → 무조건 도망친다.
  • 붙잡는다(つかまえる) → 파스칼이 도망간다. 주인공의 속도가 25이상이라면 잡아서 전투 개시
  • 이야기를 한다(はなしかける) → 파스칼이 기뻐하고 있다(パスカルは喜んでいる)의 메시지가 뜬 뒤 선택사항이 나온다.

마인 고토우 (魔人ゴトウ)

  • 상성 : 검 기술에 약하다
  • 출현 조건 : 주인공 레벨이 15이상

마인 더・히어로 (魔人ザ・ヒーロー)

  • 출현 조건 : 주인공 레벨이 25이상, 파티 속성이 뉴트럴, 선택지에서 YES를 선택해야 메시지가 변화

마인 로우 히어로 (魔人ロウ・ヒーロー)

  • 상성 : 파마, 주살, 신경 무효
  • 출현 조건 : 주인공 레벨이 25이상, 파티 속성이 카오스, 선택지에서 YES를 선택해야 메시지가 변화

마인 카오스 히어로 (魔人カオス・ヒーロー)

  • 상성 : 파마, 주살, 신경 무효
  • 출현 조건 : 주인공 레벨이 25이상, 파티 속성이 로우, 선택지에서 YES를 선택해야 메시지가 변화

마인 엔노오즈노 (魔人エンノオズノ)

  • 상성 : 파마, 주살무효, 마법 흡수/검, 총, 기술에 약하다
  • 출현 조건 : 주인공 레벨이 35이상

마인 앨리스 (魔人アリス)

  • 상성 : 검 기술에 약함
  • 출현 조건 : 주인공 레벨이 30이상, 월령이 만월일 때 마법의 상자를 열면 낮은 확률로 마인 앨리스가 출현한다. 이후 보석을 주지 않는 선택지를 고르면 전투를 할 수 있다.
