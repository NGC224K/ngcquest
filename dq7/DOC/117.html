몬스터 직업/전직 조건

  1. 설명
  2. 중급직 전직 조건
  3. 상급직 전직 조건

설명

  • 몬스터 직업의 기본직은 몬스터의 마음이 필요하다. 마음을 소지한 상태에서 <다마 신전>의 전직 화면을 보면 해당 몬스터 직업이 있다. (참고 : 몬스터의 마음 입수법)
  • 중급직과 상급직은 '몬스터의 마음'으로도 전직할 수 있고, 인간 직업과 동일하게 마스터(★8) 직업 조합으로도 전직이 가능하다.
  • 에빌 에스타크(エビルエスターク), 데스 머신(デスマシーン)은 '몬스터의 마음'이 없다. 따라서 하위 직업 조합으로 전직하는 방법만 가능하다.

중급직 전직 조건

직업 조합으로 전직하는 방법
중급직 조건
안드레알 썬더래트 + 게류온
저주의 램프 썩은 시체 + 미믹
갸오스 담비라무초 + 립스 + 에빌 터틀
헬 배틀러 춤추는 보석 + 플라잉 데빌
코스모 팬텀 우물 마인 + 립스
드라고 슬라임 슬라임 + 리자드맨
플라잉 데빌 키메라 + 에빌 터틀 + 꽃물총새
게류온 리자드맨 + 담비라무초
사신 귀족 썩은 시체 + 호이미 슬라임 + 춤추는 보석
프로토 킬러 골렘 + 썬더래트
골렘 폭탄암 + 버서커
우물 마인 버서커 + 미믹

상급직 전직 조건

직업 조합으로 전직하는 방법
상급직 조건
다크 비숍 사신 귀족 + 코스모 팬텀
마인 부두 저주의 램프 + 헬 배틀러
로즈 배틀러 우물 마인 + 꽃물총새
기가 뮤턴트 갸오스 + 안드레알
무지개 공작 플라잉 데빌 + 로즈 배틀러
플라티나 킹 에빌 에스타크 + 드라고 슬라임
에빌 에스타크 안드레알 + 데스 머신
데스 머신 골렘 + 프로토 킬러
