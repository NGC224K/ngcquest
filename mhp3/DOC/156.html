육질 (물리)

모든 부위 중에서 제일 효과가 높은 부위.
공격이 성립되는 범위내에서 단순하게 데미지가 크게 들어가는 부위를 말함.
※ 실제로는 노리기 어려운 부위도 있으므로 자세한 육질은 상세 일람을 확인해주세요.

기절 은 효과 시간으로 랭크를 정함
　장:15초 이상　중:10초 이상　단:10초 미만　－:효과 없음

내성
3회 상태이상을 시키기 위해 필요한 축적치의 평균을 기준으로 랭크로 정함.

함정·구속 아이템

○: 효과 있음···효과 시간에 관계 없이, 효과가 있는 경우.
△: 조건부로 효과 있음···분노나 잠수 등 특수한 조건이 있는 경우.
×: 효과 없음···전혀 효과가 없는 경우.

섬광옥과 함정, 단발식구속탄에는 내성이 존재한다. 효과 시간은,
대형 몬스터 : 내성 축적치와 현재 스태미너(피로 유무)로 정해진다.
아마츠마가츠치 등 스태미너가 존재하지 않는 대형 몬스터 : 내성 축적치로 정해진다.

  • 구멍함정 : 몬스터를 구멍에 떨어뜨려 일정 시간 구속한다. 효과 중 주는 데미지 1.1배
  • 마비함정 : 몬스터를 마비시켜 일정 시간 구속한다.
  • 섬광옥 : 몬스터의 눈을 멀게 한다.
  • 악취구슬 : 위험도가 낮은 몬스터를 다른 에리어로 이동시킨다. 또는 구속 공격에서 즉시 이탈한다.
  • 음폭탄 ：음폭탄·철갑유탄·통폭탄 등의 음파 효과나 격룡선의 큰 징이 해당한다.
  • 고기함정 :고기·독고기·마비고기·수면고기가 해당. 각각 200 축적 + 스태미너 회복

특수


※ 감기
해당 몬스터의 감기 내성을 나타낸다. 감기 무효의 몬스터는 피로 상태가 되지 않는다.
스태미너의 최대치나 행동마다의 스태미너 소비량에 영향을 받으므로 피로 상태가 되기 쉬움에는 직결하지 않는다.
　◎:200 이하　○:300 이하　△:301 이상　－:효과 없음

※ 포효
해당 몬스터가 사용하는 포효의 위력
  • 소→ 포효【소】. 방어하려면 「이전」또는 「고급이전」이 필요.
  • 대→ 포효【대】. 방어하려면 「고급이전」이 필요.
  • 특→ 포효【대】에 충격파가 첨부.

※ 풍압
해당 몬스터가 사용하느 풍압의 위력
  • 소→풍압【소】. 방어하려면 「풍압무효」가 필요.
  • 대→풍압【대】. 방어하려면 「풍압무효」또는 「풍압완전무효」가 필요.
  • 특→특수한 효과를 가진 풍압

비고

해당 몬스터가 특정 상태이상을 일으키는 공격 행동을 하는 것을 나타낸다.
  • 독 : 독 및 맹독 상태. 해독제·한방약 등으로 치유 가능. 스킬「독무효」로 무효화 가능
  • 마비 : 마비 상태. 스킬「마비무효」로 무효화 가능
  • 수면 : 수면 상태. 휘청거릴 때 원기드링크로 즉시 해소 가능. 스킬「수면무효」로 무효화 가능
  • 기절 : 맞으면 확실히 기절하는 공격을 가짐. 스킬「기절무효」로 무효화 가능
  • 진동 : 휘청거리는 효과가 있는 지진. 스킬「내진」로 무효화 가능
  • 악취 : 악취로 인하여 음식 아이템을 사용 불가. 소취옥으로 즉시 해소 가능. 스킬「향균」으로 무효화 가능
  • 진흙 : 진흙 투성이 상태. 소산제로 즉시 해소. 스킬「니&설무효」로 무효화 가능
  • 눈 : 눈사람 상태. 소산제로 즉시 해소. 스킬「니&설무효」로 무효화 가능
  • 방 : 방어력 다운 (소20%/대50%). 인내의씨·경화약계로 즉시 해소.  「철면피」스킬로 무효화 가능
  • 내 : 속성내성 다운 (소10/대30). 소거의씨로 즉시 해소.
  • 속성에 당함(속성 상태이상), 소거의씨로 즉시 해소.
    ○내성【소】·【대】 스킬로 무효화 가능
    회피 행동으로 효과 시간 감소
    • 화 :화속성 당함【소/대】. 15초/20초간 지속 데미지 (소는 2초마다, 대는 1초마다 2데미지)
    • 수 :수속성 당함【소/대】. 60초간 스태미너 회복 속도 감소 (소는 1/2, 대는 1/4)
    • 뇌 :뇌속성 당함【소/대】. 60초간 받는 기절치 2배
    • 빙 :빙속성 당함【소/대】. 60초간 스태미너 감소 속도 증가 (소는 2배, 대는 3배)
