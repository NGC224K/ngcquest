개요

  • 기본적으로 보병계, 기병계, 궁병계의 병과가 있다.
  • 신설, 보충 비용은 각 도시의 시설(XX편제소, 기술개발원)이나 도시의 종류(방목/기병)에 따라 변동된다. 해당 병과에 관련된 시설이 있으면 값이 싸진다.
  • 자동 전투시 부대가 전멸하는 일은 거의 없으면서 경험치는 입수 가능하다.
  • 보충으로 병력이 늘어나면 경험치와 사기가 떨어진다. 만일 강화할 수 있는 경험치가 모였지만 보충으로 내려갔다면 다시 경험치를 쌓아야 한다.
  • 공성전에서 방어측이라면 궁병계로 많은 양의 경험치를 획득할 수 있다.

보병계

  • 모든 병과의 기본. 상대가 궁병계 부대라면 매우 강력하다.
  • 신설ㆍ보충에 드는 비용이 가장 싸기 때문에 NPC 태수는 보병계를 우선해 정비하는 경향이 있다.

보병

  • 신설 가능 도시 : 모든 도시
  • 신설 비용ㆍ보충 비용 : 1000 ㆍ500
  • 특징 : 운제(성벽에 사다리를 놓고 올라가는 것) 가능

중보병

  • 강화 가능 도시 : 보병편제소가 있는 도시
  • 강화 조건 : 보병의 경험이 300이상
  • 보충 비용 : 600~900
  • 특징 : 운제 가능. 보병에 비해 공격력이 높다.

근위병

  • 강화 가능 도시 : 보병편제소와 기술개발원이 있는 도시 (남피, 허창, 강주, 완, 강하)
  • 강화 조건 : 중보병의 경험이 500이상
  • 보충 비용 : 700~1050
  • 특징 : 운제 가능. 공격력ㆍ방어력 모두 높다.

만족병

  • 신설 가능 도시 : 만병편제소가 있는 도시 (무릉, 건안, 교지)
  • 신설 비용ㆍ보충 비용 : 2000ㆍ1000~1500
  • 특징 : 운제 가능. 공격력은 중보병 수준, 방어력은 보병과 동등. 산지나 삼림에서의 이동력이 높다. 독천에 데미지를 받지 않는다.
  • 비고 : 야전에서는 기병 같은 수준의 기동력으로 이동 가능하다.

등갑병

  • 신설 가능 도시 : 등갑편제소가 있는 도시(건녕, 운남)
  • 신설 비용ㆍ보충 비용 : 1200ㆍ600~900
  • 특징 : 운제 가능. 모든 병과 중 최고의 방어력. 독천으로 데미지를 받지 않는다. 화계에 의한 데미지가 크다
  • 비고 : 방어력이 높기 때문에 「궁시」로는 좀처럼 데미지가 주어지지 않는다. 천적은 「목수」
    • NPC 태수에 건녕이나 운남을 맡기면 이 부대만을 신설한다.

청주병

  • 신설 가능 도시 : 신설 불가
  • 보충 비용 : 1000
  • 특징 : 운제 가능. 보병계 최고의 공격력. 방어력도 높다.
  • 비고 : 입수 수단은 이하 세 가지 방법.
    • 조조(혹은 조조 군의 무장)로 플레이. 사실 이벤트 「조조의 대두」를 발생시킨다. 3000명 × 10부대가 손에 들어 온다.
    • 조조군 이외로 플레이중이라면 「조조의 대두」가 발생한 직후, 복양의 태수를 임면 하기 전에 공격해서 도시를 획득 (2~3 부대 획득 가능)
    • 조진이 사병으로서 가지고 있을 때 조진을 등용한다. (시나리오에 따라 다름)

기병계

  • 기병계만으로 편성된 군세는 전체 MAP상에서 행군 속도가 빠르다. 또 도시나 거점에 대한 공격력이 높다. (자동 전투시)
  • 공격의 수단이 기본적으로 「근접」밖에 없기 때문에 공격시에는 손해가 반드시 따른다.
  • 「지면」에서만 행동 가능하고 성벽으로 이동이나 배치를 실시할 수 없다.

기병

  • 신설 가능 도시 : 모든 도시
  • 신설 비용ㆍ보충 비용 : 700~1400ㆍ350~700
  • 특징 : 이동력이 높다.

중기병

  • 강화 가능 도시 : 기병편제소가 있는 도시 (양평, 북평, 진양, 상당, 안정, 서평, 무위)
  • 강화 조건 : 기병의 경험이 300이상
  • 보충 비용 : 400~1200
  • 특징 : 이동력이 높다. 기병에 비해 공격력이 높다.

호표기

  • 강화 가능 도시 : 기병편제소와 기술개발원이 있는 도시 (진양, 안정, 서평)
  • 강화 조건 : 중기병의 경험이 500이상
  • 보충 비용 : 500~1350
  • 특징 : 이동력이 높다. 중기병보다 더욱 공격력이 높다. 방어력은 기병계 최고
  • 비고 : 조조의 친척(하후돈ㆍ하후연ㆍ하후패ㆍ조휴ㆍ조순)이 재야일 때 사병으로서 가지고 있다.

돌기병

  • 신설 가능 도시 : 돌기편제소가 있는 도시 (북평, 상당, 무위)
  • 신설 비용ㆍ보충 비용 : 2000ㆍ1000~1500
  • 특징 : 이동력이 높다. 「궁시, 화시」가능 (사정거리 2~4). 공격력은 중기병, 방어력은 기병과 동등.
  • 비고 : 북방의 이민족 무장이 사병으로서 가지고 있다.

상병

  • 신설 가능 도시 : 상병편제소가 있는 도시 (영창)
  • 신설 비용ㆍ보충 비용 : 3000ㆍ1500~2250
  • 특징 : 이동력은 기병계 중 최하(보병 수준). 공격력은 전 병과 중 최상이다. 「복병」커맨드 사용 불가. 불이 붙으면 폭주 상태에 빠진다.
  • 비고 : 불에 폭주 상태가 되면 조작 불능이 되어 랜덤으로 이동한다.
    폭주 상태가 되었을 경우 폭주 유닛의 주변이나 진로상에 있는 부대를 적,아군 따지지 않고 공격한다.

궁병계

  • 수동 전투시는 가옥(기후가 「안개」)이나 삼림을 공격 목표로 하는 것이 가능.
  • 보병계나 기병계와 달리 특수 병과가 없다.
  • 타격에 약하기 때문에 부대끼리의 난투에 참가시키거나 궁병VS궁병의 사격전이 되지 않게 하는 것이 중요.
  • 이동ㆍ사격에 걸리는 행동력이 크기 때문에 정란을 배치하지 않으면, 원융노병까지 강화하는 것이 상당한 고생이지만 원융 노병이 된 부대는 공성전의 주역이 된다.

궁병

  • 신설 가능 도시 : 모든 도시
  • 신설 비용ㆍ보충 비용 : 1200ㆍ600
  • 특징 : 운제 가능. 「궁시」 「화시」가능(사정거리 2~3)

노병

  • 강화 가능 도시 : 궁병편제소가 있는 도시
  • 강화 조건 : 궁병의 경험이 300이상
  • 보충 비용 : 700~1050
  • 특징 : 운제 가능. 「궁시」 「화시」가능(사정거리 2~4). 공격력이 궁병보다 높다.

원융노병

  • 강화 가능 도시 : 궁병편제소와 기술개발원이 있는 도시 (한중, 무도, 완, 강하)
  • 강화 조건 : 노병의 경험이 500이상
  • 보충 비용 : 800~1200
  • 특징 : 운제 가능. 「궁시」 「화시」가능(사정거리 2~4). 공격력은 노병과 같다.
    다만 사격 공격에 관통 속성이 있다. 일부 지형이나 성문 안에 있는 부대에도 데미지를 준다.
