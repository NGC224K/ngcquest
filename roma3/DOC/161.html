로아누 (ロアーヌ)

출현 방법 1. 오프닝 이벤트를 종료한다
2. 카타리나의 경우 마스카레이드(マスカレイド)를 되찾으면 들어갈 수 있다
동료 1. 오프닝 이벤트 직후만 술집에서 토마스, 하리드, 사라, 엘렌을 동료로 가능(미카엘, 모니카, 카타리나의 경우 불가)
2. 주인공이 유리안일 경우 모니카의 '프린세스 가드'를 거절하면 위의 멤버를 동료로 할 수 있다
메인 이벤트 뷰네이의 소굴의 어비스 게이트를 닫는다
서브 이벤트 없음
정보 수집 1. 술집 마스터로부터 정보를 듣는다 → 뮬스 출현
2. 남서쪽 민가의 노인에게 정보를 듣는다 → 타프탄 산 출현
