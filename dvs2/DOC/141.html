분기 조건의 기본 설명

스토리 분기

6TH DAY를 마지막으로, 스토리가 4개로 분기한다.
진행된 루트에 의해서 아래의 요소가 변화한다.
  • 7TH DAY의 스토리 전개
  • 영입 가능 캐릭터
  • 라스트 보스의 제2단계에서 싸우는 상대
  • 엔딩의 내용

루트 진출 조건에 대하여

다이치, 야마토, 로나우도 루트는 특별한 조건이 없어도 선택에 의해서 갈 수 있다.
다만, 염려하는 자 루트는 특정 조건을 만족해야 진행할 수 있다. (조건은 아래에서 설명)

동료 캐릭터에 대해서

  • 각 루트의 초기 멤버의 캐릭터는 인연 스테이지와 관계없이 생존하고 있다면 주인공과 합류한다.
  • 초기 멤버가 아닌 캐릭터는 인연 스테이지 4 이상으로 설득이 가능. 3 이하라면 설득이 실패한다.
    • 예외로 염려하는 자 루트에서만, 다이치와 이오는 인연과 상관없이 반드시 동료가 된다.
      이때 이오가 사망한 상태라면, 대신 히나코가 인연 스테이지와 상관없이 반드시 동료가 된다.
  • 로나우도는 <야마토 루트 or 염려하는 자 루트>로 진행하면, 반드시 사망한다.
  • 염려하는 자는 <염려하는 자 루트> 이외로 진행하면, 반드시 사망한다.

다이치 루트

초기 멤버 : 다이치, 이오, 히나코, 쥰고

난이도는 중간 정도, 루트 A라면 라스트 보스의 전투에서 조금 어려울지도 모른다.

다이치 루트는 LAST DAY "나아갈 길"의 이벤트 선택사항으로 2개로 분기한다.
  • 다이치 루트 A 선택법 : 선택사항으로 "세계를 복원한다.(世界を復元する)"
  • 다이치 루트 B 선택법(조건 있음) : 선택사항으로 "폴라리스를 쓰러뜨린다.(ポラリスを倒す)"

추가로 염려하는 자를 제외한 12명 전원을 동료로 영입시킨 상태에서 다이치 루트로 엔딩을 볼 경우 대단원(大団円)을 볼 수 있다. (A, B 무관)

다이치 루트 B의 조건

B 루트로 진행하려면, 이전에 몇 가지 이벤트를 진행해야 한다.
  • 3RD DAY : "니카이아의 수수께끼" → "서버는 어디다 (サーバーは何処だ)"
  • 4TH DAY : "악마 고찰" → "정보의 소재" → "터미널의 원리"
  • 5TH DAY : "적대의 진의"
  • 6TH DAY : "예정 조화 (予定調和)" → "개념의 세계"에서 아카식 레코드의 존재를 알아낸다. (선택사항 있음)
  • 7TH DAY : 후미를 설득 → "가역적인 발상 (可逆的な発想)"
상기 이벤트를 진행하면, "폴라리스를 쓰러뜨린다."의 선택지가 출현한다.

아카식 레코드의 선택사항
아래의 선택사항을 올바르게 선택하면 분기 플래그를 달성할 수 있다.
うなってる? → そんなに驚くな → そうなだ → 言ってた → ないでしょ → 完全なる無の話 → なるほど → 分かった →
しますぅ… → それだ! → ポラリスだ → 書き変える気だ → ポラリスを利用する → そうだ → あるかもな →
言った → なるほどね → 参ったな.

야마토 루트

초기 멤버 : 마코토, 야마토, 케이타, 후미

난이도가 가장 쉬운 루트다. 1주차 플레이 시, 선택하면 좋다.

로나우도 루트

초기 멤버 : 죠, 아이리, 로나우도, 오토메

다이치 루트와 비슷한 난이도

염려하는 자 루트

초기 멤버 : 염려하는 자

모든 루트 중에 난이도가 가장 높다.

'염려하는 자'의 루트는 염려하는 자의 인연 스테이지 4 이상으로 해두면, <6TH DAY 23:30>에 "새로운 세계"가 출현한다.
여기의 이벤트에서 올바른 선택사항을 선택하면 <염려하는 자 루트>로 진행할 수 있다. (틀리면 루트 소멸)
何してる → 謝るな → それはいけない事? → 悪じゃないよ → そうだ → ポラリスを倒したら? →
今すぐ考えろ → 操ってくれ → どうなるんだ? → そうだ.
