찾아가는 방법

금강신계에 가는 방법

  • 게임을 클리어 후 다시 플레이하면 금강신계에 갈 수 있습니다. (LOAD로도 가능)
  • 가는 방법은 旧신주쿠(서) 동쪽의 뮤턴트 남자와 대화하면 됩니다.


다른 장소에 가는 방법

속성 진입 조건 위치 진입 조건 위치

  • 금강신계에서 다른 장소로 워프가 가능한데 주인공의 속성과 능력치에 따라 워프되는 장소가 다릅니다.
  • 각 장소에서 최고의 무구 획득이 가능합니다.
  • 스테이터스 체크를 피하는 방법이 있습니다. "금강신계의 스테이터스 체크 피하는 방법"을 참고하세요.

금강신계


  • 小 : 엔노오즈노
  • 箱 : 청동의 상자 6개
  • ★1 : 아마테라스
  • ★2 : 히루코
  • ★3 : 이자나미 (HP/MP 회복)
  • ★4 : 마인 출현 포인트 (GBA판에서는 버그 때문에 마인이 1종. 마타도르(マタドール)만 출현합니다.)
  • 知25 : 주인공의 지혜가 25이상 필요
  • 体25 : 주인공의 체력이 25이상 필요
  • 魔25 : 주인공의 마력이 25이상 필요
  • 運25 : 주인공의 운이 25이상 필요
  • N : 뉴트럴 속성이면 열린다.
  • C : 카오스 속성이면 열린다.
  • L : 로우 속성이면 열린다.
  • W1 : 旧황궁 워프
  • W2 : 旧이차가야 워프
  • W3 : 旧도쿄 타워 워프
  • V : 콜로세움 입구 (GBA판 전용)

금강신계의 콜로세움에서 우승하면 좋은 아이템을 받을 수 있습니다.
이 중에는 메모리 칩과 향, 보석, 고급 도구 등을 많이 받을 수 있고 강력한 무기와 방어구도 나옵니다. 꼭 해보세요.
