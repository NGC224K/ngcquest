능력치 상승

  • 시스템LIOH : 기력 130에 발동. 격투·기량·명중 +20
  • 재밍 기능 : 자기소대 전원의 명중률, 회피율 +10%
  • 마징파워 : 기력 130에 발동. 최종 데미지 +25%.
  • 제로 시스템 : 기력 140에 발동. 탑승한 파일럿의 명중·회피 +20, 격투·사격·방어·기량 +10

특수 활성화

  • 분신 : 기력 130 이상으로 발동. 50%확률로 적의 공격을 회피
  • 무트론 개방 : 기력 140으로 발동. 라이딘의 숨겨진 무기가 사용 가능.
  • 폭주 : 격추할 때 발동, 컨트롤 불가.

회복

  • HP회복(소~대) : 아군 턴에 HP최대치의 10%(소), 20%(중), 30%(대)를 회복.
  • EN회복(소~대) : 아군 턴에 EN최대치의 10%(소), 20%(중), 30%(대)를 회복.
  • S2기관 : 플레이어 페이즈에 EN최대치의 50%를 회복.

장비

  • 변형 : 기체를 변형시킴.
  • 분리 : 기체를 분리.
  • 분리(単) : 분리 이전 상태에서 격추 당하면 분리 상태가 된다.
  • 합체 : 2기 이상의 유닛을 합체하여 새로운 모습이 된다.
  • 검 장비 : 특수기능 '베어내기' 발동 조건
  • 방패 장비 : 특수 기능 '쉴드방어' 발동 조건
  • 수리장치: '수리(이동 후 사용 가능)' 커맨드 사용 가능. 소대 전체의 HP를 회복한다. 수리장치가 있는 기체가 포함된 소대는 매 턴 HP이 10% 회복.
  • 보급장치 : '보급(기본적으로 이동 후 사용 불가)' 커맨드 사용 가능. 탄과 EN을 회복한다. 보급장치가 있는 기체가 포함된 소대는 매 턴 EN이 10% 회복.
  • XX 블록 : 해당 타입의 특수 효과를 무효화. 직격(정신기)를 사용하면 블록 효과를 무효화하고 특수 효과가 걸린다.
